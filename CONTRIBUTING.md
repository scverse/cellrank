# Contributing to CellRank

We welcome contributions! Please see our
[contributing guide](https://cellrank.readthedocs.io/en/latest/contributing.html)
for details on setting up a development environment, running tests, writing
documentation, and our PR workflow.

If you have questions, open an [issue](https://github.com/theislab/cellrank/issues/new/choose)
or reach out via [Discourse](https://discourse.scverse.org/c/ecosystem/cellrank/).
